## 📋 简介

为网络工程师量身打造的华为（Huawei）和华三（H3C）交换路由设备命令集支持扩展。提供丰富的命令提示、代码片段、语法高亮和自动更新功能，极大提升网络设备配置效率。

**优化版 v1.3.0**：模块化架构、全面去重、性能优化、600+条唯一命令
